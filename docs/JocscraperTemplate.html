Class: JocscraperTemplate

JocscraperTemplate(targetWebsite, regionNames, xPathVariations, xPathTotalAdverts, numberRegex, pageTimeout)

Class `JocscraperTemplate` serves as an interface that all scrapers inherit from. It provides common methods and properties needed for scraping job listings from various websites.

Constructor

new JocscraperTemplate(targetWebsite, regionNames, xPathVariations, xPathTotalAdverts, numberRegex, pageTimeout)

Constructor initializes the scraper with specific configurations.
Parameters:
Name Type Description
targetWebsite string The base URL of the target website to scrape.
regionNames Map A map of region names to their corresponding URL paths.
xPathVariations Array An array of objects defining different path variations for scraping.
xPathTotalAdverts string XPath selector for the total number of adverts.
numberRegex RegExp Regular expression to extract numbers from text.
pageTimeout number Timeout value for page loading in milliseconds.
